# C++ Modules - 42 Lisboa 💻

A series of projects designed to introduce and master **Object-Oriented Programming (OOP)** concepts using C++.

## 📝 Overview
This repository contains my solutions for the C++ modules at **42 Lisboa**. The goal is to move beyond procedural C and learn how to design robust, scalable, and reusable software using the C++98 standard (as required by the 42 curriculum).

## 🚀 Learning Path

### [Modules 00 - 04]: The Basics of OOP
- **Module 00:** Namespaces, classes, member functions, and stdio streams[cite: 3].
- **Module 01:** Memory allocation, pointers to members, and references[cite: 3].
- **Module 02:** Ad-hoc polymorphism, operator overloading, and floating-point basics[cite: 3].
- **Module 03:** Inheritance and memory management in class hierarchies[cite: 3].
- **Module 04:** Subtype polymorphism, abstract classes, and interfaces[cite: 3].

### [Modules 05 - 09]: Advanced Concepts
- **Module 05:** Exception handling (Try/Catch blocks)[cite: 3].
- **Module 06:** C++ Casts (scalar conversion)[cite: 3].
- **Module 07:** Templates (generic programming)[cite: 3].
- **Module 08:** STL (Standard Template Library) containers and algorithms[cite: 3].
- **Module 09:** Complex data structures and performance optimization[cite: 3].

## 🛠️ Technical Constraints
- **Standard:** C++98[cite: 3].
- **Compiler:** `c++` with flags `-Wall -Wextra -Werror -std=c++98`.
- **Guidelines:** Adherence to the **Orthodox Canonical Class Form** (Constructor, Copy Constructor, Assignment Operator, Destructor).
